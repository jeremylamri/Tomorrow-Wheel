# Guide Administrateur - Tomorrow Wheel V3

## Accès
Cliquez sur l'icône **Paramètres (Rouage)** en haut à droite de l'écran. Le panneau s'ouvre avec un fond flouté.

## Fonctionnalités

### 1. Contrôles du Jeu
- **Mode VIP :** 
  - Cliquez sur le bouton "Mode VIP".
  - La liste des lots s'illumine.
  - **Cliquez sur un lot** pour le sélectionner (Badge "FORCE" apparaît).
  - *Comportement :* Le prochain lancer tombera obligatoirement sur ce lot. Le moteur physique s'adaptera pour que cela paraisse naturel.
- **Reset Mémoire :** 
  - Efface l'historique des gains et les numéros déjà tirés.
  - Ne supprime PAS la configuration des lots.
  - Une confirmation est demandée.

### 2. Configuration Tirage
- **Nombre Maximum :** Définit la borne haute pour le tirage aléatoire (ex: Si 150 invités, mettez 150).
- **Test RNG (Diagnostic) :**
  - Cliquez sur le bouton "Activity/Test RNG".
  - Le système simule instantanément 1000 tirages.
  - Une alerte affiche les statistiques (Moyenne, Répartition par tranche).
  - Si "✅ DISTRIBUTION SAINE" s'affiche, le hasard est équitable.

### 3. Gestion des Lots
- **Ajout :** Champ texte en bas du panneau.
- **Suppression :** Icône poubelle.
- **Note :** Les changements sont sauvegardés dans la mémoire de la session (perdus si rafraîchissement de page, sauf si persistance ajoutée futurement).

## Astuces de Présentation
- **Plein Écran :** Utilisez l'icône flèches (ou F11) pour masquer l'interface du navigateur.
- **Son :** Assurez-vous que le système de son de la salle supporte les basses fréquences pour que le "Drone" d'ambiance soit efficace.
